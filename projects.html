MULTI-LABEL NEURAL NETWORK FOR POLYPHONIC SOUND EVENT DETECTION

Mentored by Dr. Dinkar Sitaram, Director Center for Cloud Computing and Big Data and Ms. Savitha Murthy, Assistant Professor, Computer Science and Engineering.
  • Duration: Aug ’15 – May ‘16
  • Place: Bangalore, India

Proposed a sound event detection system trained using a minimally annotated data set of single sounds to identify and separate the components of polyphonic sounds.
The system uses a Feed Forward Neural Network pre-trained using auto-encoders.
Single sounds, represented as Mel Frequency Cepstral Coefficient (MFCC) feature vectors, are used to train the neural network using back-propagation.
Polyphonic sounds are preprocessed using Principal Component Analysis and Non-negative Matrix Factorization (NMF) to obtain source separated sounds.
The source separated sounds are then tested for sound classification using the feed-forward algorithm.
The system was able to achieve a reasonable accuracy of source separation and detection with minimal training set using state of the art evaluation metric.
